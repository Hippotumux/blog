例子

我們想要用一條線分類 O X ，之後將會利用基於 sigmoid function 的神經網路來找。

sigmoid funcion

Sigmoid function 長如下

σ(x)=11+e−x\sigma(x) = \frac{1}{1+e^{-x}} σ(x)=1+e−x1​

微分形式

σ′(x)=σ(x)(1−σ(x))\sigma'(x) = \sigma(x)(1-\sigma(x)) σ′(x)=σ(x)(1−σ(x))

可以改變為位移，和縮放，例如 σ(3(x−5))\sigma(3(x-5))σ(3(x−5)). 右移到 5 然後更加陡峭。

可以利用 sigmoid function 設定精神元層，如果一個神經元為一個向量 aaa，其輸出到下一層則長成如下

σ(Wa+b)\sigma(Wa + b) σ(Wa+b)

  • WWW 是一個矩陣，代表權重
  • bbb 是一個向量，代表偏差

寫出分量則是

σ(∑jwijaj+bi)\sigma(\sum_j w_{ij}a_j + b_i) σ(j∑​wij​aj​+bi​)

為了方便不會用分量來顯示。

The General Set-up

上面提到的四層神經網路例子，會稱 Layer1 為 Input layer，Layer4 為 Output layer，中間的層則叫做 Hidden layer。 每一層的向量被稱為神經元 Neurons。

現在來轉換成一般形式，假設總共有 LLL 層神經網路，第 111 層為輸入層，第 LLL 層為輸出層，假設第 lll 層為 nln_lnl​ 神經元，整個流程就是從 Rn1→Rnl\mathbb{R}^{n_1} \rightarrow \mathbb{R}^{n_l}Rn1​→Rnl​

我們用 W[l]∈Rnl×nl−1W^{[l]} \in \mathbb{R}^{n_l \times n_{l-1}}W[l]∈Rnl​×nl−1​ 定義在 lll 層的權重矩陣。而 wjk[l]w^{[l]}_{jk}wjk[l]​ 就是在 lll 層的神經元 jjj 應用於第 l−1l-1l−1 層神經元 kkk 輸出的權重。 同樣的 b[l]∈Rnlb^{[l]} \in \mathbb{R}^{n_l}b[l]∈Rnl​ 就是第 lll 層的偏差。

Stochastic Gradient

我們的目標是 minimize CostCostCost function，用最一開始的四層神經元例子，假設狀態權重和偏差那些參數總和為 ppp，也就是說 p∈R23p \in \mathbb{R}^{23}p∈R23，或者一般化為 p∈Rsp \in \mathbb{R}^sp∈Rs，則 Cost:Rs→RCost: \mathbb{R}^s \rightarrow \mathbb{R}Cost:Rs→R。

gradient descent (GD)

或稱為 steepest desecent，此方法是用迭代的方式進行，目標為讓其收斂到最小化 CostCostCost。

假設目前的向量為 ppp，下一個向量為增加一個擾動 Δp\Delta pΔp，也就是 p+Δpp + \Delta pp+Δp 我們要如何選擇 Δp\Delta pΔp 來使迭代有效果呢?

如果 Δp\Delta pΔp 很小，則 ∥Δp∥\lVert \Delta p \rVert∥Δp∥ 小到可以忽略，所以利用泰勒展開可以得到

Cost(p+Δp)≈Cost(p)+∑r=1s∂Cost(p)∂prΔprCost(p+\Delta p) \approx Cost(p) + \sum_{r=1}^s \frac{\partial Cost(p)}{\partial p_r} \Delta p_r Cost(p+Δp)≈Cost(p)+r=1∑s​∂pr​∂Cost(p)​Δpr​

或者用 gradient ∇\nabla∇ 符號來表示

Cost(p+Δp)≈Cost(p)+∇Cost(p)TΔpCost(p+\Delta p) \approx Cost(p) + \nabla Cost(p)^T\Delta p Cost(p+Δp)≈Cost(p)+∇Cost(p)TΔp

透過 Cauchy-Schwarz inequality，假設兩個 f,g∈Rsf, g \in \mathbb{R}^sf,g∈Rs 我們有 ∣fTg∣<=∥f∥2∥g∥2|f^Tg| <= \lVert f \rVert_2 \Vert g\Vert_2∣fTg∣<=∥f∥2​∥g∥2​ 所以最負值 ∣fTg∣|f^Tg|∣fTg∣ 為 −∥f∥2∥g∥2-\lVert f \rVert_2 \Vert g\Vert_2−∥f∥2​∥g∥2​ 且 f=−gf = -gf=−g

所以既然要最小化，我們的 Δp\Delta pΔp 就要選擇 −∇Cost(p)-\nabla Cost (p)−∇Cost(p)

然而我們有了方向，但是實際移動的不能太多，所以需要一個 step size ttt 或稱為學習率

也就是說

p→t∇Cost(p)p \rightarrow t\nabla Cost(p) p→t∇Cost(p)

會執行直到超出迭代次數，或者達到停止標準 (和預期答案足夠靠近)

回顧

我們的 CostCostCost 定義為

Cost=1N∑i=1N12∥y(x{i})−a[L](x{i})∥22Cost = \frac{1}{N} \sum_{i=1}^{N} \frac{1}{2} \lVert y(x^{\{i\}}) - a^{[L]}(x^{\{i\}}) \rVert_2^2 Cost=N1​i=1∑N​21​∥y(x{i})−a[L](x{i})∥22​

每個分量上的 CostCostCost 為

Cx{i}=12∥y(x{i})−a[L](x{i})∥22C_{x^{\{i\}}} = \frac{1}{2} \lVert y(x^{\{i\}}) - a^{[L]}(x^{\{i\}}) \rVert_2^2 Cx{i}​=21​∥y(x{i})−a[L](x{i})∥22​

所以取 gradient 後為

∇Cost(p)=1N∑i=1N∇Cx{i}(p)\nabla Cost(p) = \frac{1}{N} \sum_{i=1}^N \nabla C_{x^{\{i\}}}(p) ∇Cost(p)=N1​i=1∑N​∇Cx{i}​(p)

但是會發現，如果有大量參數，或者大量點，這樣會導致做一次 gradient 計算成本非常昂貴。

stochastic gradient descent (SGD)

一種很便宜的替代方案是每次都隨機選擇一個點的梯度，替換掉所有點上的平均值，這也是隨機梯度的最簡單形式。方法如下

  1. 從{1,2,...N}\{1,2,...N\}{1,2,...N} 選擇一個"均勻隨機"的值 iii
  2. 更新 $$p \rightarrow t\Delta C_{x^}(p)$$

或者我們也可以隨機時不取後放回，也就是隨機一輪，這也稱為完成一個 epoch。

  1. 從{1,2,...N}\{1,2,...N\}{1,2,...N} 打亂成新的排序 {k1,k2,k3,...,kN}\{k_1, k_2, k_3, ..., k_N\}{k1​,k2​,k3​,...,kN​}
  2. iii 從 111 到 NNN 更新 $$p \rightarrow t\Delta C_{x^}(p)$$

問題: 如果學習率太大，容易造成參數更新呈現鋸齒狀的更新，這是很沒有效率的路徑。

mini-batch stochastic gradient descent

把訓練集分割成比較小的， 一堆小小的資料集，我們稱他們稱為 "mini-batches" 。當你訓練非常多的資料直到收斂為止的過程，小批次梯度下降跑得比批次梯度下降還快得多。

所以假設 m<Nm < Nm<N，也就是一次不取 N ，而是一個小批次 m 。

  1. 從 {k1,k2,k3,...,km}\{k_1, k_2, k_3, ..., k_m\}{k1​,k2​,k3​,...,km​} 均勻隨機從 1∼N1 \sim N1∼N 取
  2. 更新 $$p \rightarrow t \frac{1}{m} \sum_{i=1}^m\Delta C_{x^}(p)$$

Levenberg-Marquardt algorithm (LM)

能提供數非線性最小化（局部最小）的數值解。此演算法能藉由執行時修改參數達到結合高斯-牛頓算法以及梯度下降法的優點，並對兩者之不足作改善，比如高斯-牛頓算法之反矩陣不存在或是初始值離局部極小值太遠

pk+1=pk+(JTJ+μI)−1[JT(y−F(pk))]p^{k+1} = p^k + (J^TJ + \mu I)^{-1}[J^T(y - F(p^k))] pk+1=pk+(JTJ+μI)−1[JT(y−F(pk))]

  • J 為 Jacobian
  • 如果 μ\muμ 很小，其實就是高斯牛頓的方法
  • 反之，就是最速下降

可以注意到因為 JJJ 為 m×nm \times nm×n 其中 mmm 為資料點， nnn 為所有 parameter 的數量，JTJ∈Rn×nJ^TJ \in \mathbb{R}^{n \times n}JTJ∈Rn×n，通常情況下 mmm >> nnn，但如果遇到 nnn >> mmm 該如何解決呢？

Woodbury matrix identity

在數學中（特別是線性代數領域），Woodbury 矩陣恆等式以 Max A. Woodbury 命名，該公式表明，對某個矩陣進行秩為 𝑘 的修正，其逆矩陣可以通過對原始矩陣的逆矩陣進行秩為 𝑘 的修正來計算。這個公式的其他名稱包括矩陣反演引理（Matrix Inversion Lemma）、Sherman–Morrison–Woodbury 公式或僅稱為 Woodbury 公式。

Woodbury 矩陣恆等式的具體形式為:

(A+UCV)−1=A−1U(C−1+VA−1U)−1VA−1(A + UCV)^{-1} = A^{-1}U(C^{-1} + VA^{-1}U)^{-1} V A^{-1} (A+UCV)−1=A−1U(C−1+VA−1U)−1VA−1

所以套用在 L-M 上，假設

A=λIA = \lambda IA=λI, U=JTU = J^TU=JT, B=IB = IB=I, V=JV = JV=J ，所以得到

(λI+JTJ)−1=λ−1I−λ−1JT(λI+JJT)−1Jλ−1( \lambda I + J^T J)^{-1} = \lambda^{-1}I - \lambda^{-1}J^T(\lambda I + JJ^T)^{-1} J \lambda ^{-1} (λI+JTJ)−1=λ−1I−λ−1JT(λI+JJT)−1Jλ−1

會發現我們成功把 JTJJ^TJJTJ 改為 JJTJJ^TJJT

Back propagation (BP)

中文稱反向傳播，是對多層類神經網路進行梯度下降的演算法，也就是用鏈式法則 (chain rule) 以網路每層的權重為變數計算損失函式的梯度，以更新權重來最小化損失函式。
